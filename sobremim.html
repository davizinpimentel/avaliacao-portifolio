    HTML
    O HTML é uma linguagem de marcação. Com ele, podemos marcar os elementos utilizando tags. As tags servem para dar contexto ao conteúdo, realçar parte do texto que queremos diferenciar dos demais. Então, tags são utilizadas para dar definição aos conteúdos.

    CSS
    CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site.
